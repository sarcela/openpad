# Contributing to OpenPad

Thanks for helping improve OpenPad 💙

## Quick start

1. Fork this repo
2. Create a branch from `main`
3. Make focused changes
4. Open a PR with context + screenshots (if UI)

## Development notes

- Main app sources live in `OpenPad/`
- Keep PRs small and easy to review
- Prefer reversible changes
- For iPad UI changes, include before/after screenshots

## Commit style

Use clear, scoped messages, for example:

- `fix(chat): send on return and shift+return newline`
- `feat(attachments): improve PDF/image extraction`

## Pull request checklist

- [ ] Builds successfully in Xcode
- [ ] No secrets/tokens in code
- [ ] Updated docs when behavior changed
- [ ] Added screenshots for visible UI changes

## Reporting bugs

Please include:

- Device/iPadOS version
- Expected vs actual behavior
- Steps to reproduce
- Logs or screenshots if possible
